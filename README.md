## üìñ Project Title & Description

**F1 Merchandise Web System** is a simple web-based system developed to provide a login and registration interface for users before accessing the main dashboard. The system is designed with a modern user interface and smooth transitions between login and registration forms using JavaScript.

This project serves as a front-end prototype for an authentication system in a web application.

##  Features Included

* User Login interface
* User Registration interface
* Toggle animation between Login and Register forms
* Password input protection
* Redirect to dashboard page after successful login
* Responsive layout for different screen sizes

##  Instructions to Test Login

Follow the steps below to test the login functionality:

1. Open the `index.html` file using a web browser
2. The **Login form** will be displayed by default
3. Enter any username and password (no validation applied ‚Äì front-end only)
4. Click the **Submit** button
5. The system will automatically redirect to `dashboard.html`
